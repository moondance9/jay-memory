- 머니클럽 가입 조건에서 ‘머니트리카드 등록’ 요건 삭제

- 개편 적용 목표 시점: 8월 중
- 머니트리사업팀 담당 실무자 지정 요청
- 첨부: 가입 프로세스 개선 및 등급제 개편안(.docx), API 요청사항(.xlsx), 관련 이미지
