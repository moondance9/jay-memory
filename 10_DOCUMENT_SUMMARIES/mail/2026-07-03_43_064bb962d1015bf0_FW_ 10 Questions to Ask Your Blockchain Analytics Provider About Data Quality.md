# 핵심 요약
Chainalysis Weekly에서 발송한 블록체인 분석 데이터 품질 관련 10가지 질문과 평가 기준을 공유하는 이메일입니다.

# 신규 내용
- 블록체인 분석 제공자의 데이터 품질을 평가하기 위한 10가지 핵심 질문 제시
- 데이터 출처, 소유권 추론 방식, 법적 기준 충족 여부, 증거 기반 결론 도출 방법 등 주요 검토 포인트 안내
- 신뢰할 수 있는 분석 제공자는 클러스터 구성 및 근거를 명확히 설명할 수 있어야 함
- 관련 링크를 통해 상세 분석 자료 제공

# 중복 / 인용 / 포워딩
- 이전 Chainalysis 관련 뉴스레터 및 분석 자료 인용 포함 (약 5,148자)
- Telegram 알림에서 접힌 상태로 제공됨

# 관련 프로젝트 추정
- 블록체인 데이터 분석 및 컴플라이언스 관련 프로젝트
- 내부 또는 외부 블록체인 분석 제공자 평가 및 선정 작업

# 주요 사실
- 데이터 품질이 컴플라이언스, 규제, 조사 결과에 직접적 영향
- 잘못된 데이터는 조사 실패 및 부당한 고객 해지 초래 가능
- 법 집행 기관 데이터와 익명 제보 데이터의 차이점 강조
- 머신러닝 결과와 증거 기반 결론 구분 필요

# 첨부파일
- 총 6개의 이미지 파일 (jpg) 첨부, 내용 미지원

# 리스크 / 쟁점
- 데이터 품질 미흡 시 조사 및 컴플라이언스 리스크 발생
- 분석 방법의 투명성 및 법적 기준 충족 여부가 핵심 쟁점

# 액션아이템
- Chainalysis가 제시한 10가지 질문을 바탕으로 현재 분석 제공자 평가
- 내부 컴플라이언스 팀과 공유하여 데이터 품질 검증 절차 강화
- 필요 시 Chainalysis 상세 분석 자료 검토 및 참고

# 관계자
- 김주형 (발신자)
- Chainalysis Weekly (정보 제공자)
- 내부 컴플라이언스 및 조사팀
