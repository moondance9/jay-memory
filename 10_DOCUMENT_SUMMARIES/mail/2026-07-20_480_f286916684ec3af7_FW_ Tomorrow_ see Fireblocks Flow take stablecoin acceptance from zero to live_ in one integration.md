# 핵심 요약
Fireblocks에서 단일 SDK 통합으로 스테이블코인 수용 과정을 완성하는 라이브 데모를 7월 21일 오전 11시(ET)에 진행한다는 안내 메일.

# 신규 내용
- 스테이블코인 수용은 지갑 연결, 변환, 컴플라이언스, 정산을 포함하는 복합 기능임.
- Fireblocks Flow는 이 모든 과정을 하나의 SDK로 해결.
- 7월 21일 오전 11시(ET)에 Adam Zion(제품 책임자)과 Eric Tesenair(솔루션 엔지니어)가 라이브 데모 진행.
- 데모 내용: 실시간 입금 흐름, 800개 이상 외부 지갑 및 거래소 연결, 자동 변환, 사용자별 컴플라이언스, 감사 가능한 정산.
- 라이브 참석이 어려워도 등록 시 녹화본 제공.

# 중복 / 인용 / 포워딩
- 과거 회신 및 포워딩 본문 약 2,279자 접힘 처리.

- 800개 이상의 외부 지갑 및 거래소와 연결 가능.
- 컴플라이언스 및 정산 기능 포함.
