# 핵심 요약
해외 여권 진위확인 연동을 위한 개발 API 문서와 서버 구성, 방화벽 오픈 대상, 반입 장비 모델명 등 구체적인 연동 환경 안내가 포함된 신규 메일이다.

# 신규 내용
- 해외여권 진위확인 API는 기존 신분증 진위확인 API와 동일한 경로(/api/reqidv)를 사용하며, 별도 서버 증설 없이 기존 IDV_API 서버에 통합 구성됨.
- 모니터링과 통계도 기존 관리 화면(IDV_WEB)에서 통합 관리.
- 아웃바운드 HTTPS 443 포트 오픈 대상은 PROD1, PROD2, DEV 서버에서 auth.useb.co.kr와 api3.useb.co.kr 두 호스트.
- OAuth 토큰 발급과 해외여권 진위확인 API 호출 경로 및 용도 명시.
- 반입 장비 모델명: 노트북 950QDB, 960XGK.
- 문의사항 발생 시 연락 요청.

# 중복 / 인용 / 포워딩
- 과거 회신 및 포워딩 본문 약 1,986자 포함되어 있으나 접혀 있음.
- 이전 메일에서 해외 여권 진위확인 연동 가이드, 방화벽 요청, 노트북 모델명 요청, 개발서버 OS 및 환경 변경 안내 포함.

# 관련 프로젝트 추정
- 갤럭시아머니트리 해외 여권 진위확인 연동 개발 프로젝트
- 인지소프트와 협업하는 신분증 및 여권 진위확인 API 통합 관리

- 기존 신분증 진위확인 API와 동일 서버 및 경로 사용
